# Privacy Policy

Effective date: 2026-07-01

YouTube Grid Customizer does not collect, transmit, sell, or share personal information.

The extension stores user settings such as grid column count, search result column count, keyword filters, case sensitivity, Shorts visibility preference, and Game Room visibility preference in Chrome extension storage. These settings are used only to provide the extension's layout customization and filtering features on YouTube pages.

The extension runs on YouTube pages so it can adjust the page layout and hide videos that match user-defined keywords. It does not send YouTube page content, browsing activity, or user settings to the developer or to any external server.

The extension does not use analytics, advertising, tracking, accounts, or remotely hosted executable code.

If this policy changes, the updated policy will be published with a new effective date.
